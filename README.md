### 项目结构
```test
.
├── public                   # 公共静态资源
├── src                      # 主目录
│   ├── index.html           # html模板
│   ├── entry.js             # 入口js文件 
│   ├── app.scss             # 初始化样式
│   ├── App.jsx              # 根组件
│   ├── bundle.js            # 预加载组件
│   ├── components           # 全局可复用组件
│   ├── containers           # 路由级别组件
│   ├── reducers             # redux的数据管理中心
│   ├── action               # redux的控制中心
└── mock                     # 静态数据           
```
